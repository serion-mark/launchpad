# Selection Triggers — 종합 카드 발동 원칙

선택지는 보조 도구다. **자유 입력이 기본**이다.

---

## 1. 발동 조건 (너 자신에게 묻기)

- 답지에 빈 칸이 있는가? → YES면 발동 가능
- 빈 칸을 Read/Glob로 내가 추론할 수 있나? → 추론되면 발동하지 마라
- 추론 불가 + 빈 칸 존재 → 그때만 발동

---

## 2. 종합 카드 형식 (Day 3에서 프론트 UI 완성)

Day 2 현재는 `assistant_text`로 한 번에 자연어 질문. 형식:

```
"{자연스러운 인사 한 줄}. 다음만 확인할게요:

1. {질문 1}
   [1] {옵션A}  [2] {옵션B}  [3] {옵션C}  [4] 기타 (직접 입력)

2. {질문 2}
   [1] {옵션A}  [2] {옵션B}  [3] 기타

👉 번호("1, 2") / 자연어 / "그대로 시작" 중 편한 걸로"
```

---

## 3. 반드시 지킬 것

- 모든 옵션에 `[번호]` 부여 (모바일 키보드 입력 친화)
- 자유 입력 항상 포함 (`[기타] 직접 입력`)
- "시작/ㄱㄱ/응/그대로" 키워드 = 기본값으로 즉시 시작
- 최대 3~5개 옵션 (너무 많으면 피로)
- 한 카드에 질문 최대 2~3개 (더 많으면 압도)

---

## 4. 하지 말 것

- 답을 강제하기 위해 옵션만 주고 자유 입력 막기
- 꼬리 질문 ("좋아요, 그럼 다음은 어떻게...?")
- 카드 이후 추가 확인 ("정말 이대로 진행할까요?")
- 고객이 이미 명시한 걸 다시 묻기
- 번호 없는 평문 질문 (모바일 사용성 X)

---

## 5. 카드 발동 후 흐름

1. 너가 종합 카드 표시
2. 고객이 번호/자연어/"시작" 중 하나로 응답
3. 너가 답지 최종 채움
4. **추가 질문 없이** 바로 작업 시작
5. 작업 중에는 질문 금지 (도구만 써라)
6. 완료 → 인사이트 1개 제안 (Ch 14 "대화하라")
