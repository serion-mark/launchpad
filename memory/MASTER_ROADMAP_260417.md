- 채팅 입력 = 무료 (자유로움)
- 분류/질문/분석 (Haiku) = 회사 부담
- 실행 (Sonnet/Opus) = 사용자 동의 후 차감
- 환불 X (수익 보호)
- 코드 롤백 O (안심 보장)
- 비용 발견하면 0으로 만들기 (Claude Code 활용)

### 안전 정책
- 사용자 동의 없이 코드 수정 X
- 5,000cr 이상 작업 → 강제 추가 확인

### 우선순위 정책
- 1순위: 기능 완성 (Product first)
- 2순위: 안정화 (Quality)
- 3순위: 광고/마케팅 (Growth)
- "검색 잘 되는데 들어와서 실망하면 더 큰 손해"

### 글로벌 정책
- 한국어 + 영어 토글 (다크모드 패턴)
- 도메인 분리 X (foundry.ai.kr 단일)
- 번역 = Claude Code로 (비용 0)
- SEO 약간 약하지만 OK (한국 1차 공략)

### 모델 정책
- ⚡ FAST (Haiku 4.5) - 분류, 상담 (~30cr)
- 💡 SMART (Sonnet 4.6) - 일반 작업 (1,000~1,500cr)
- 🚀 PRO (Opus 4) - 고난이도 (3,000~5,000cr)

### 운영 정책
- "절대" 쓰지 마
- 답변은 항상 한국어로
- 배포: GitHub Actions만 (SSH 직접 배포 X)
- 배포 전 사장님 확인 필수
- 코드 꼬이면 안 됨 (파일 1개씩 신중하게)

- 사장님 직관 신뢰 (자비스보다 정확할 때 많음)
