### 1. 홈 — `/`

레이아웃: 와이즐리 스타일 — 프로모 배너 + 카테고리 + 상품 그리드

**상단 헤더**
- 로고: "스마트몰" (굵은 보라색)

**프로모션 배너 (큰 카드)**

- 서브: "신규 회원 한정, 3일간만!"
- [지금 쇼핑하기 →] 버튼

**카테고리 아이콘 (가로 스크롤)**

|--------|------|
| 🏆 | 베스트 |

| 💊 | 비타민 |
| 🏋️ | 프로틴 |
| 🌿 | 다이어트 |
| 🫧 | 피부 |
