## 배경 — 왜 이 작업을 하는가

### Foundry 포지셔닝 (확정)
```
"Foundry로 MVP 만들고, 외주사·개발자 미팅에 가세요.
 견적이 절반으로 줄어듭니다."
```

- Foundry = 외주 대체가 아니라 **외주 미팅 전 필수 무기**
- MVP가 기획서 역할 → 외주사가 바로 이해 → 견적 절반 + 소통 제로
- 아이디어→검증→MVP→미팅 무기까지 **원스톱**

### 세일즈 킬링 포인트 (모두의 창업 타겟!)
```
"200만원 지원금, 똑똑하게 쓰세요.
Foundry 15만원에 MVP 만들고,
남은 185만원으로 마케팅·CS·디자인 하세요.
앱이 있어야 마케팅을 하든 CS를 붙이든 하죠!"
```
- Foundry = 다른 AI SaaS 쓰기 전 "1순위 필수재"
- 경쟁이 아니라 공생 (파운더리 결제 → 나머지로 뤼튼/채널톡/미리캔버스)

#### 섹션 2: 3단계 프로세스

```
외주사 미팅 전, 이 3단계만 거치세요.
```

**Step 1: AI가 사업을 검증합니다**
- AI 3개(Claude·GPT·Gemini)가 각자 관점에서 분석
- 시장성 / 경쟁사 / 리스크 / 예산 배분까지
- (AI 회의실 실제 스크린샷)
- 참고: 동네키친 회의 결과 스크린샷 사용

**Step 2: 30분 만에 MVP가 완성됩니다**
- 검증 끝나면 "이걸로 앱 만들기" 클릭
- AI가 코드 생성 → 자동 배포 → 내 URL 생성
- (빌더 화면 스크린샷 — 파일 생성 중 + 프리뷰)

**Step 3: 이걸 들고 미팅에 가세요**
- "이거 기반으로 해주세요" → 외주사가 바로 이해
- 견적 절반 + 소통 비용 제로 + 실패 방지
- (비교 아이콘: 말로 설명 vs 화면 보여줌)

#### 섹션 5: 외주 vs Foundry 비교표

| | 외주 개발 | Foundry |
|------|----------|---------|
| 비용 | 3,000만원~ | **15~30만원** |

| 수정 | 변경비 수백만원 | **클릭 한 번** |
| 실패 | 완성 후 발견 | **사전 검증** (AI 회의실) |

디자인: 왼쪽(외주) 회색/빨간색, 오른쪽(Foundry) 파란색/초록색 강조

#### 섹션 7: 자주 묻는 질문 (FAQ)

**Q. 진짜 코딩 몰라도 돼요?**
A. 네. 타자만 치면 됩니다. 클릭으로 수정하고, 채팅으로 기능 추가하세요.

**Q. 만든 앱으로 실제 사업이 가능한가요?**
A. 네. 실제 작동하는 URL이 생성되고, 코드 다운로드도 가능합니다.

**Q. 외주를 맡기려면 이걸 왜 써요?**
A. MVP를 들고 미팅에 가면 견적이 절반으로 줄어듭니다. 화면을 보여주면 소통 비용이 사라집니다.

**Q. 정부지원사업비로 결제 가능한가요?**
A. 네. 모두의 창업, 예비창업패키지 등 정부지원금으로 사용 가능합니다.

**Q. AI 회의실이 뭔가요?**
A. Claude, GPT, Gemini 3개 AI가 사장님의 아이디어를 각자 관점에서 분석하고 토론합니다. 시장성, 경쟁사, 리스크, 예산 배분까지 2분이면 끝납니다.
