## 페이지 구성 (3페이지)

### 1. 홈 (시작) — `/`

레이아웃: 중앙 정렬 히어로 + 시작 버튼

**상단**: 달+별 이모지 🌙✨
**타이틀**: "오늘 밤, 잘 자고 계신가요?"
**부제**: "5가지 질문으로 나의 수면 건강을 체크하고 AI 맞춤 조언을 받아보세요"
**시작 버튼**: [수면 체크 시작하기 →] (크고 눈에 띄게)

**하단 통계 카드 3개**:
| 카드 | 값 |
|------|-----|
| 총 진단 수 | 12,847명 |
| 평균 수면 점수 | 64점 |
| 개선 사례 | 89% |

### 2. 진단 (5단계) — `/check`

레이아웃: **스텝 프로그레스바 + 질문 1개씩 + 선택지**

상단에 프로그레스바: ● ● ○ ○ ○ (현재 단계 표시)

**Step 1: 취침 시간**
"보통 몇 시에 잠자리에 드시나요?"
선택지 (카드 버튼):
- 🌅 22시 이전
- 🌙 22시~24시
- 🌃 24시~02시
- 🦉 02시 이후

**Step 2: 기상 시간**
"아침에 보통 몇 시에 일어나시나요?"
선택지:
- ☀️ 6시 이전
- 🌤 6시~8시
- ⛅ 8시~10시
- 😴 10시 이후

**Step 3: 카페인**
"하루에 커피나 카페인 음료를 몇 잔 드시나요?"
선택지:
- 🚫 안 마심
- ☕ 1~2잔
- ☕☕ 3~4잔
- ☕☕☕ 5잔 이상

**Step 4: 스크린 타임**
"잠들기 전 스마트폰/TV를 얼마나 보시나요?"
선택지:
- ✅ 안 봄
- 📱 30분 이내
- 📱📱 30분~1시간
- 📱📱📱 1시간 이상

**Step 5: 운동**
"일주일에 운동을 얼마나 하시나요?"
선택지:
- 🚶 거의 안 함
- 🏃 1~2회
- 💪 3~4회
- 🏋️ 5회 이상

각 스텝에서 선택하면 자동으로 다음 단계로 (애니메이션 전환)
[이전] [다음] 버튼도 있음

**Step 5 선택 후 → "AI가 분석 중..." 로딩 2초 → 결과 페이지로**

### 3. 결과 — `/result`

레이아웃: 점수 원형 + 분석 카드 + 팁 리스트

**수면 점수 (크게 중앙)**
- 원형 게이지 (CSS): 72점 / 100점
- 등급: "양호 😊" (색상: 하늘색)
- 등급 기준: 0~40 위험(빨강) / 41~60 주의(노랑) / 61~80 양호(파랑) / 81~100 최적(초록)

**분석 카드 5개 (각 질문별)**

| 항목 | 선택 | 평가 | 한 줄 코멘트 |
|------|------|------|------------|
| 취침 시간 | 24시~02시 | ⚠️ 주의 | 이상적 취침 시간은 22~23시입니다 |
| 기상 시간 | 8시~10시 | ✅ 양호 | 7~8시간 수면은 적절합니다 |
| 카페인 | 3~4잔 | ⚠️ 주의 | 오후 2시 이후 카페인을 줄여보세요 |
| 스크린 타임 | 1시간 이상 | 🔴 위험 | 블루라이트가 멜라토닌 분비를 억제합니다 |
| 운동 | 1~2회 | ✅ 양호 | 규칙적인 운동이 수면 질을 높입니다 |

**AI 맞춤 조언 3개**
1. 💡 "취침 시간을 23시로 앞당겨보세요. 2주간 시도하면 체내 시계가 조정됩니다."
2. 💡 "오후 2시 이후 디카페인으로 전환하면 수면 질이 30% 이상 개선됩니다."
3. 💡 "잠들기 1시간 전 스마트폰 대신 종이책을 읽어보세요. 멜라토닌 분비가 촉진됩니다."

**하단 버튼**: [다시 체크하기] [결과 공유하기]

## 네비게이션

이 앱은 피드/대시보드와 다르게 **단방향 플로우**:
홈 → 진단(5스텝) → 결과

**헤더만 존재** (사이드바/하단탭 없음):
- 로고 "꿀잠체크 🌙" + [다시 하기] 링크

## 공통 컴포넌트
- `Header.tsx` — 로고 + 홈 링크
- `ProgressBar.tsx` — 스텝 프로그레스 (1~5)
- `QuestionCard.tsx` — 질문 + 선택지 카드 버튼
- `ScoreGauge.tsx` — 원형 점수 게이지 (CSS)
- `AnalysisCard.tsx` — 항목별 분석 결과
- `TipCard.tsx` — AI 조언 카드

## 핵심 인터랙션 (state 관리)
```
const [step, setStep] = useState(0)        // 0=홈, 1~5=질문, 6=로딩, 7=결과
const [answers, setAnswers] = useState({})  // {sleep:'24-02', wake:'8-10', ...}
```
- 선택 시 → answers 업데이트 + step +1
- step 6 → 2초 setTimeout → step 7 (결과)
- 결과 점수는 answers 기반으로 하드코딩 계산 (AI API 호출 X)
- 전체가 **단일 페이지(page.tsx)에서 step으로 전환**해도 OK (간단하니까)
