### Foundry 메인 페이지 MVP 메시지 (포트폴리오 리빌드 시 같이 반영!)

**핵심 카피** (확정):
> "Foundry로 MVP 만들고, 외주사·개발자 미팅에 가세요.
>  견적은 반으로, 소통 비용은 제로로."

**배치 위치**: foundry.ai.kr 메인 랜딩 + 포트폴리오 페이지 상단/하단

**왜 이 메시지가 강력한가**:
- 도전자 90%가 "한 줄 아이디어"만 있고 기획서/화면설계 없음 (딥리서치 결과)
- MVP 없이 외주사 미팅 → 소통 비용 폭발 + 견적 3,000만원+ + 기획 증발
- MVP 들고 미팅 → "이거 기반으로 해주세요" → 견적 반, 소통 제로, 실패 방지
- Foundry = 외주 대체가 아니라 **외주 미팅 전 필수 무기**로 포지셔닝

**구체적 효과 3가지** (포트폴리오 또는 가격표 페이지에 추가):
1. 💰 **견적 50% 절감** — MVP가 기획서 역할, 외주사가 바로 이해
2. 🗣️ **소통 비용 제로** — 말로 설명 X, 화면 보여주면 끝
3. 🛡️ **실패 방지** — 완성 전에 고객 반응 먼저 테스트 가능

**홈페이지 섹션 구성 제안**:
```
[히어로] "아이디어만 말하면, AI가 전략부터 MVP까지"
   ↓
[포트폴리오] 실제 앱 9개 (LIVE 클릭 체험)
   ↓
[MVP 가치] ⭐ "MVP 만들고, 외주사·개발자 미팅에 가세요"
           견적 반 / 소통 제로 / 실패 방지
   ↓
[가격] 라이트 49만 / 스탠다드 99만
   ↓
[CTA] "무료로 시작하기"
```
