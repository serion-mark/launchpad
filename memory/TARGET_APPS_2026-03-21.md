## ⭐ Foundry 핵심 철학 (사장님 원문 — 3/21 대화)

> "AI의 가능성은 무한한데, 일반 사용자는 그 한계를 정해놓고 사용하지.
> 우리는 그 무한한 영역을 연결해주는 도구라고 생각해!"

> "우리의 고객은 AI 전문가가 아니야. AI를 채팅으로 사용하는 사람들을
> 나같은 전문가처럼 만들어주고 돈을 받는 서비스를 하는 거잖아?"

### 이 철학이 제품에 미치는 영향

**문제**: custom 템플릿 하나로 모든 앱을 만들 수 있다 → 하지만 고객은 그걸 모른다
- 일반인은 "AI로 앱을 만든다"는 개념 자체가 낯설음
- "아무거나 만들 수 있어요"라고 하면 오히려 뭘 해야 할지 모름
- 고객은 스스로 가능성에 한계를 정해버림 ("이런 건 안 되겠지...")

**해결**: 구체적인 예시를 많이 보여줘서 가능성을 열어줘야 함
- 포트폴리오 10개: "아, 이런 것도 만들 수 있구나!"
- 업종별 템플릿 카드: "내 업종이 있네!" → 진입장벽 낮춤
- 챗봇: "이런 앱 만들 수 있나요?" → "네, 만들어 드릴게요" → 바로 빌더로

### Foundry의 역할 재정의

```
일반인이 보는 AI:        "ChatGPT에 질문하는 것"
├ 한계를 정해놓음:       "코딩은 안 되겠지"
├ 가능성을 모름:         "앱을 만든다고?"
└ 두려움:               "나는 개발 못 하는데..."

Foundry가 하는 일:       "무한한 AI를 일반인의 손에"
├ 가능성을 보여줌:       "이런 앱도 만들 수 있어요" (포트폴리오)
├ 전문가 수준으로:       "질문만 답하면 전문가처럼" (가이드형 질문지)
├ 결과물을 줌:           "진짜 작동하는 앱" (코드+배포)
└ 소유권까지:            "코드는 100% 당신 것" (다운로드)
```

### 마케팅 메시지에 반영

**Before (개발자 관점)**:
> "AI가 Next.js + Supabase 앱을 자동 생성합니다"

**After (사장님 철학 반영)**:
> "개발자 없이도, 여러분의 사업 아이디어를 작동하는 앱으로 만들어 드립니다"
> "AI가 어렵다고요? 질문에 답하기만 하면 됩니다"
> "3천만원짜리 개발을 30만원에. 나머지는 사업에 투자하세요"

### 제품 UX에 반영할 사항
1. **"자유롭게 만들기"보다 → 업종별 카드를 먼저 보여주기** (선택의 역설 방지)
2. **포트폴리오를 10개 이상** 다양하게 → "이런 것도 되나?" 의문을 사전에 해소
3. **챗봇 상담** → "이런 앱 만들 수 있나요?" 질문에 "네!" 하고 바로 빌더 연결
4. **사용 후기/케이스 스터디** → "나도 개발 모르는데 만들었어요" 사회적 증거

## 소셜/매칭 앱 — 니치 시장 상세

### 사장님이 좋다고 한 아이템들
1. **취미 기반 매칭** — 등산/러닝/독서/보드게임 등 취미로 사람 연결
2. **지역 동호회 매칭** — 동네 기반 소모임/동호회 가입+활동
3. **시니어 만남** — 5060 세대 친구/동반자 매칭 (블루오션!)

### Supabase로 구현 가능한 기능 구조
```
profiles     — 사진, 소개, 취미, MBTI, 지역
likes        — 좋아요/스킵 기록
matches      — 서로 좋아요 = 매칭 성사
messages     — 1:1 채팅 (Supabase Realtime)
reports      — 신고/차단
groups       — 동호회/모임 (선택)
events       — 오프라인 모임 일정 (선택)
```

### 왜 정부지원사업에서 잘 나오는 아이템인가
- "기존 앱은 20~30대 중심, 시니어용이 없다" → 차별점 명확
- "우리 동네 취미 동호회 앱이 없다" → 지역 특화
- 심사위원이 좋아하는 키워드: "사회적 고립 해소", "시니어 디지털 포용"
- 정부지원금 3,000~5,000만원으로 외주 맡기면 MVP도 못 만듦 → Foundry 30만원 해결

## 1차 타겟 고객 규모 (연간)

| 지원사업 | 선정 인원/팀 | Foundry 잠재 고객 |
|---------|------------|----------------|
| 모두의창업 (지역+테크) | ~3,000명 | 앱 만들고 싶은 30% = ~900명 |
| 예비창업패키지 | ~5,000명 | 앱 필요 40% = ~2,000명 |
| 초기창업패키지 | ~3,000명 | 앱 필요 50% = ~1,500명 |
| 재도전 성공패키지 | ~2,000명 | 앱 필요 30% = ~600명 |
| **합계** | **~13,000명** | **~5,000명 (잠재)** |

5,000명 중 전환율 2%만 잡아도 = **100명 x 30만원 = 3,000만원/년**
전환율 5% = **250명 x 30만원 = 7,500만원/년**
+ 호스팅 MRR: 100명 x 9,900원/월 = **월 99만원 (연 1,188만원)**
