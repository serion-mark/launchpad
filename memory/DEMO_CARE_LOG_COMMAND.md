## 카페노트와 차별화 포인트
둘 다 B. 대시보드형이지만:
- 색상: 갈색(커피) vs **초록(돌봄)** — 완전 다른 분위기
- 데이터: 매출/재고 vs **어르신 건강/일정** — 완전 다른 도메인
- 타임라인: 카페노트에 없는 **시간순 일정 타임라인** 추가
- 알림: 카페노트에 없는 **긴급도별 건강 알림** 추가
- 느낌: "사업 관리" vs **"사람 돌봄"** — 감성이 다름
