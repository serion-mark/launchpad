## 사업 방향 전환 (핵심!!)

### 포지셔닝
- AS-IS: "앱을 만들어드립니다" → 완성품 기대 → 실망
- TO-BE: "MVP 프로토타입 + 개발 미팅 도구"
- 타겟: 창업자, 기획자, PM, **정부지원사업 합격자**
- 가치: "기획서 100장보다 작동하는 앱 1개"

### 핵심 인사이트: 사업계획서 → AI 회의실 → MVP
정부지원사업 합격자는:
1. 사업계획서가 이미 있다 (시장분석, 타겟, 기능, 수익모델 포함!)
2. 사업비가 있다 (돈 있음!)
3. MVP를 만들어야 한다 (필요함!)
4. 개발자를 찾아야 한다 (급함!)
5. 정산 가능해야 한다 (세금계산서!)

→ 사업계획서를 AI 회의실에 넣으면 3개 AI가 분석
→ 분석 결과 기반으로 앱 생성
→ 훨씬 정확한 MVP가 나옴!
→ 이걸 가지고 개발자/외주사 미팅!
= 완벽한 Product-Market Fit!!

### 고객 여정 2가지
A. 아이디어만 있는 사람: /start 채팅 → 설계 → MVP
B. 사업계획서 있는 사람: AI 회의실 → 분석 → MVP (추천!)
