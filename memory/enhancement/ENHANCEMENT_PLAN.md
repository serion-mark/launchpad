## 🟡 단기 (1~2주 내)

### 7. ⭐⭐⭐ 추천 개선사항 투명한 카드 UI — Foundry 핵심 가치!

**[비즈니스 의미 — 사장님 핵심 철학]**
> "한 번에 완벽한 앱이 아니라, 반복 개선으로 점점 좋아지는 앱"
> 이게 Foundry의 차별화 포인트 (Lovable, Bolt 대비 강점)

**[현재 약점 2가지]**
1. 추천 개선사항 = 파일명만 나열 → 비개발자는 클릭 못 함
   - "src/app/components/streak-counter.tsx" ← 이게 뭔지 모름
   - "누르면 뭐가 바뀌는데? 잘못 누르면 1,000cr 날아가는데..."
2. 채팅으로 기능 변경 불가 (편집 모드만 가능) — 별도 #17번 항목

**[해결: 투명한 카드 UI]**
파일명만이 아니라 카드로 보여주기:
- 📊 변경 내용 미리보기 (한국어, 비개발자 친화적)
- 🔒 변경 안 되는 것도 명시 (안심)
- 🎨 카테고리 분류 (디자인/⚡기능/🔧품질/📊데이터)
- 💰 비용 + ⏱️ 예상 시간
- 🔄 마음에 안 들면 1-클릭 롤백 (versions 활용, 코드만 롤백)
- 다른 방향 재시도 옵션 (추가 1,000cr → 추가 수익!)

**[카드 UI 예시]**
```
┌─────────────────────────────────────────┐
│ 🎨 연속 복용 카운터 디자인 개선         │
│ 📍 streak-counter.tsx                   │
│                                          │
│ 변경 내용:                               │
│  ✓ 숫자 애니메이션 추가                 │
│  ✓ 마일스톤 표시 (7/30/100일)           │
│  ✓ 색상 개선 (회색 → 그라데이션)        │
│                                          │
│ 변경 안 되는 것:                         │
│  ✓ 기능 그대로 (카운팅 동작)            │
│  ✓ 데이터 안전                          │
│  ✓ 다른 페이지 영향 없음                │
│                                          │
│ 💰 1,000cr  ⏱️ 30초                     │

└─────────────────────────────────────────┘

[개선 후]
✅ 개선 완료!
[변경 내용 보기] [다른 방향으로 시도] [이전으로 되돌리기]
```
